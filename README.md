# Deskripsi
Deret Fibonacci adalah barisan bilangan dimana setiap suku merupakan hasil penjumlahan dua suku sebelumnya.
contoh: 0, 1, 1, 2, 3, 5, 8, 13, ...

# Cara Kerja Program
- Program meminta input nilai n dari user
- Menggunakan fungsi rekursif untuk menghitung Fibonacci
- Menampilkan deret dari suku ke-0 sampai ke-n

# Konsep yang digunakan
- Rekursif
- Perulangan (loop)
